---
type: component
title: button
category: actions
description: Buttons help people take action, such as sending an email, sharing a document, or liking a comment.
classes:
  - name: btn
    type: component
    description: Button (Defaults to a primary filled button)
  - name: btn-tonal
    type: modifier
    description: Reduces the contrast of the button
  - name: btn-outlined
    type: modifier
    description: Removes the background of the button and adds a border
  - name: btn-elevated
    type: modifier
    description: Adds a shadow to the button and replaces the background with a neutral color
  - name: btn-text
    type: modifier
    description: Removes the background of the button
  - name: btn-info
    type: modifier
    description: Changes the color of the button to info
  - name: btn-success
    type: modifier
    description: Changes the color of the button to success
  - name: btn-warning
    type: modifier
    description: Changes the color of the button to warning
  - name: btn-error
    type: modifier
    description: Changes the color of the button to error
  - name: btn-icon-left
    type: modifier
    description: Reduces the left padding to accommodate an icon
  - name: btn-icon-right
    type: modifier
    description: Reduces the right padding to accommodate an icon
  - name: btn-icon
    type: modifier
    description: Circular button with a 1:1 aspect ratio
published: true
